# Linux開発環境構築メモ(OWASP ZAP) ※コードの表示がおかしい場合は、ソースの表示でコードを確認してください。 ## OWASP ZAP インストール ------------------------------------------------------------------------------- 1. ダウンロード https://www.owasp.org/index.php/OWASP_Zed_Attack_Proxy_Project (ZAP_2_7_0_unix.sh [linux]) 2. インストール (インストーラのあるフォルダで) $ sudo sh ZAP_2_7_0_unix.sh ※インストールはデフォルトで行う。 ※「/opt/zaproxy/」にインストールされる。 3. 「/opt/zaproxy/zap.sh」実行（admin権限） sudo sh /opt/zaproxy/zap.sh 又は 右クリックでrootで実行。 4. 画面で攻撃対象のURLを指定する。 ・クイックスタートタブの攻撃対象URLにチェックしたいURLを入れて、攻撃ボタンを押下する。

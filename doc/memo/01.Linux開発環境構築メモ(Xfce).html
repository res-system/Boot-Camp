# 01.Linux開発環境構築メモ(Xfce) ## はじめに。 ------------------------------------------------------------------------------- ※開発環境なのでセキュリティより、使いやすさを優先しています。 * ルートパス - admin * ユーザ作成 - ユーザ名:res - パスワード:res ## OSの基本設定。 ------------------------------------------------------------------------------- [参考1](http://www.xfce.org/) [参考2](http://okuman64lnx.wpblog.jp/category/centos/) [参考3](http://www.h7.dion.ne.jp/~greens/linux/de/xfce.html) ### VirtualBox設定 * 名前 linux_xfce_dev * メモリ 2048MB～4096MB * ストレージ 128GB～256GB (vdi可変) ※上記のVirtualBox作成後、vdiのコピーでも可能。 ※共有フォルダ設定は各自の環境に合わせてください。 ### CentOS7 インストール(Install CentOS7) 【VirtualBox】＞「設定」＞「ストレージ」＞「CD/DVDドライブ」→ 「CentOS-7-x86_64-Everything-1503-01.iso」を設定し「起動」する。 ### CentOS7インストール時の設定 * 言語 → 日本語 * ソフトウェアの選択 → 最小限 * インストール先 → 作成したストレージ * ネットワークとホスト名 → オンにする * ルートパス ※ルートパスは上記参照 * ユーザ作成 ※ユーザ名とパスワードは上記参照 - 管理者にチェック ### Xfceのインストール 1. epelのインストール ```ターミナル $ sudo yum install -y epel-release ``` 2. アップデート ``` $ sudo yum update -y ``` 3. X Window SystemとXfceをグループインストール ``` $ sudo yum groupinstall -y "X Window System" $ sudo yum groupinstall -y "Xfce" ``` 4. ログイン画面を出す為、モードを変更 ``` $ sudo systemctl set-default graphical.target ``` 5. インプットメソッドをインストール ``` $ sudo yum install -y gtk3-immodules ``` 6. その他必要なソフトウェアのインストール ``` $ sudo yum install -y xfce4-terminal (ターミナル) $ sudo yum install -y gedit (テキストエディタ) $ sudo yum install -y gedit-plugins $ sudo yum install -y vlgothic-fonts (フォント) $ sudo yum install -y vlgothic-p-fonts (フォント) $ sudo yum install -y firefox (ブラウザ) $ sudo yum install -y thunderbird (メーラー) $ sudo yum install -y beesu (GUI上の管理権限実行用) $ sudo yum install -y wget (wget) $ sudo yum install -y gcc (GNU Compiler Collection) $ sudo yum install -y dkms (Dynamic Kernel Module Support) $ sudo yum install -y bzip2 (圧縮解凍) $ sudo yum install -y libreoffice libreoffice-langpack-ja (office) ``` 7. 再起動 ``` $ reboot ``` ### virtualboxのguedt additionsをインストール 1. 【VirtualBoxメニュー】＞「Devices」＞「Insert Guest Additions CD Image」で CDイメージを設定する。 2. VBoxLinuxAdditionsを実行する。 ``` $ cd /run/media/[ユーザ]/VBOXADDITIONS_[バージョン]/ $ sudo ./VBoxLinuxAdditions.run $ reboot ``` ### CUI上のsudoパスワード省略 1. visudoコマンドを実行する ``` $ sudo visudo ``` 2. sudo権限を付与 ```/etc/sudoers ## Allow root to run any commands anywhere root ALL=(ALL) ALL [ユーザ名] ALL=(ALL) ALL ``` 3. パスワード不要設定 ```/etc/sudoers ## Same thing without a password # %wheel ALL=(ALL) NOPASSWD: ALL %[ユーザ名] ALL=(ALL) NOPASSWD: ALL ``` ### 共有フォルダ [参考](http://vividcode.hatenablog.com/entry/virtualbox/shared_folder) 1. VirtualBoxの共有フォルダ設定を行う。 【VirtualBoxメニュー】＞「設定」＞「共有フォルダー」で共有フォルダ設定をする。 （フォルダパスと名称は任意。自動マウントと永続化はチェックする） 2. アクセス権設定 ``` $ sudo gpasswd -a [ユーザ名] vboxsf $ reboot ``` （マウント先） /media/sf_[共有フォルダ名称] ### その他設定 * パネルはデフォルト設定を使用する。 * 【VirtualBoxメニュー】＞「View」＞「Auto-resize GuestDisplay」 * 【VirtualBoxメニュー】＞「Devices」＞「Shared Clipboard」＞「Bidirectional」 * workフォルダを作る。「/home/[ユーザ名]/work/」 * デスクトップ画像はなし。 * ファイルマネージャーの表示は詳細表示。 * 「/home/[ユーザ名]/ダウンロード/」→「/home/[ユーザ名]/download/」リネーム * firefoxのダウンロード先を「/home/[ユーザ名]/download/」に変更。 * gedit＞行番号表示＞タブを4＞VLゴシック(regular)10p * 「/home/[ユーザ名]/work/」のショートカットを作成。（送る＞サイドペイン※名称は「Work」） * 「/home/[ユーザ名]/download/」のショートカットを作成。（送る＞サイドペイン※名称は「Download」） * 「/media/sf_public/」のショートカットを作成。（送る＞サイドペイン※名称は「Public」）※共有フォルダ設定後 * 「/media/sf_public/」のショートカットを作成。（送る＞サイドペイン※名称は「Public」）※共有フォルダ設定後 * 画面左上「アプリケーションメニュー」より、メーラー(thunderbird)とブラウザ(firefox)を選択する。 * 画面左上「アプリケーションメニュー」＞「Settings」＞「外観」>フォントを『VLゴシック(regular)10p』にする。 * 日本語設定（※下記参照）。 * ファイルマネージャー(thunar)カスタマイズ（※下記参照）。 ### 不要ファイルを削除する。 ``` sudo yum clean all ``` ## 日本語環境設定 ------------------------------------------------------------------------------- [参考](http://note.kurodigi.com/centos7-ibusmozc/) 1. ファイルのダウンロード [ダウンロードサイト](http://rpm.pbone.net/) - 対象ファイル protobuf-2.5.0-4.fc19.x86_64.rpm zinnia-0.06-16.fc19.x86_64.rpm zinnia-tomoe-0.06-16.fc19.x86_64.rpm mozc-1.10.1390.102-1.fc19.x86_64.rpm ibus-mozc-1.10.1390.102-1.fc19.x86_64.rpm 2. インストール（※順番に注意） ``` $ cd /home/[ユーザ名]/download/ $ sudo yum localinstall -y protobuf-2.5.0-4.fc19.x86_64.rpm $ sudo yum localinstall -y zinnia-0.06-16.fc19.x86_64.rpm $ sudo yum localinstall -y zinnia-tomoe-0.06-16.fc19.x86_64.rpm $ sudo yum localinstall -y mozc-1.10.1390.102-1.fc19.x86_64.rpm $ sudo yum localinstall -y ibus-mozc-1.10.1390.102-1.fc19.x86_64.rpm $ reboot ``` 3. 設定 * 画面右上のキーボードの設定を「日本語入力 - Mozc」に設定する。 4. 「半角／全角」ボタンで切り替え可能になる。 ## ファイルマネージャー(thunar)カスタマイズ ------------------------------------------------------------------------------- 「ファイルマネージャー(thunar)」＞「編集」＞「アクションの設定」＞「＋」で追加する。 * tar.gzファイルを展開するコンテキストメニューを追加 - 名前: tar.gzの展開 - コマンド: tar xfvz %F - ファイルのパターン: *.tar.gz - 「その他ファイル」を選択する。 * zipファイルを展開するコンテキストメニューを追加 - 名前: zipの展開 - コマンド: unzip %F - ファイルのパターン: *.zip - 「その他ファイル」を選択する。 * root権限でファイルを実行するコンテキストメニューを追加 - 名前: rootで実行 - コマンド: beesu %F - ファイルのパターン: * - 「その他ファイル」を選択する。 * root権限でテキストファイルを開くコンテキストメニューを追加 - 名前: rootで編集 - コマンド: beesu gedit %F - ファイルのパターン: * - 「その他ファイル」を選択する。 * root権限でフォルダを開くコンテキストメニューを追加 - 名前: rootで開く - コマンド: beesu thunar %F - ファイルのパターン: * - 「フォルダー」を選択する。 * pkg.tar.xzファイルを展開するコンテキストメニューを追加 - 名前: pkg.tar.xzの展開 - コマンド: tar -Jxvf %F - ファイルのパターン: *.pkg.tar.xz - 「その他ファイル」を選択する。 * tar.bz2ファイルを展開するコンテキストメニューを追加 - 名前: tar.bz2の展開 - コマンド: tar -jxvf %F - ファイルのパターン: *.tar.bz2 - 「その他ファイル」を選択する。 ### その他アプリケーション ------------------------------------------------------------------------------- #### gnome-screenshot(ハードコピー取得用) 1. インストール ``` $ sudo yum install -y gnome-screenshot ``` 2. ショートカットキーの登録 【アプリケーションメニュー】＞「Settings」＞「キーボード」＞「アプリケーションショートカットキー」を開く。 ※保存先は「/home/[ユーザ名]」です。 * 「xfce4-screenshooter -f」(Print)を「gnome-screenshot」(Print)に変更する - コマンドをダブルクリックで変更する。 * コマンド「gnome-screenshot --window」、ショートカット(Print)で新規登録する。 ※Printが効かない？ #### bluefish(テキストエディタ) 1. インストール ``` $ sudo yum install bluefish ``` 2. ランチャーに追加する。 #### meld(ファイル比較) 1. インストール ``` $ sudo yum install meld ``` 2. ランチャーに追加する。 #### chrome(ブラウザ) 1. インストール（ダウンロードしたフォルダで実行） ``` $ sudo yum localinstall -y google-chrome-stable_current_x86_64.rpm ``` 2. ランチャーに追加する。

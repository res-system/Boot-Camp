# Linux開発環境構築メモ(Java) ## javaインストール ------------------------------------------------------------------------------- [参考](http://zero-config.com/centos/install-java8.html) 1. ファイルのダウンロード [ダウンロードサイト](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html) - 対象ファイル jdk-8u91-linux-x64.rpm 2. インストール ``` fileName.txt $ sudo yum localinstall -y /home/[ユーザ名]/download/jdk-8u91-linux-x64.rpm ``` 3. 環境変数JAVA_HOME設定 shファイル作成 ``` $ sudo vi /etc/profile.d/java.sh ``` ↓ ```java.sh #!/bin/bash JAVA_HOME=/usr/java/jdk1.8.0_91 PATH=$JAVA_HOME/bin:$PATH export PATH JAVA_HOME export CLASSPATH=. ``` 4. 実行権限を付与します ``` $ sudo chmod +x /etc/profile.d/java.sh $ reboot ``` 5. 確認 ``` $ java -version $ echo $JAVA_HOME $ echo $PATH ``` ## eclipseインストール ------------------------------------------------------------------------------- [参考](http://eclipse.az-prolog.com/install/eclipse_linux_install/) http://mergedoc.osdn.jp/ 1. ファイルのダウンロード [ダウンロードサイト](http://www.eclipse.org/downloads/) - 対象ファイル eclipse-jee-mars-2-linux-gtk-x86_64.tar.gz 2. 「/home/[ユーザ名]/work/」の下にファイルを展開 3. 日本語化 [ダウンロードサイト](http://eclipse.az-prolog.com/install/eclipse_linux_install/) - 対象ファイル pleiades_1.6.0.zip - プラグインダウンロード＞安定板＞最新版pleiades_1.6.0.zipをダウンロード 展開後、「plugins」、「features」を「/home/[ユーザ名]/work/eclipse」へコピー - eclipse.iniの修正（末尾に下記の文を追加） ```eclipse.ini -Xverify:none -javaagent:plugins/jp.sourceforge.mergedoc.pleiades/pleiades.jar ``` - ワークスペースは「/home/[ユーザ名]/work/workspace」とする。 ## eclipseの設定 ------------------------------------------------------------------------------- [参考](http://qiita.com/arai-wa/items/da807fb99a18a47b35f2) [参考](http://matome.naver.jp/odai/2135281431483912301) [参考](http://d.hatena.ne.jp/Yamashiro0217/20090616/1245121197) [参考](http://hima-zinn.hatenablog.com/entry/2015/11/22/180000) * テキスト･エディタ設定 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「一般」＞「エディター」＞「テキスト･エディタ」 2. 「表示されるタブ幅」を『2』に設定。 3. 「タブでスペースを挿入」をチェック。 4. 「空白文字の表示」をチェック。 * フォントを変更 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「一般」＞「外観」＞「色とフォント」 2. 「基本」＞「テキスト・フォント」を選択し、「編集」ボタンを押下。 3. 「VLゴシック(regular)10p」を設定。 * コードフォーマッター設定 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「Java」＞「コード・スタイル」＞「フォーマッター」 2. 「新規」を押下。 3. 「プロファイル名」に『Java開発』と入力。 4. 一般編集「タブポリシー」を『スペースのみ』に変更。※「インデント・サイズ」と「タブ・サイズ」は『4』とする。 * CSS･エディタ設定 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「Web」＞「CSSファイル」＞「エディター」 2. 「スペースを使用したインデント」を選択。 * HTML･エディタ設定 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「Web」＞「HTMLファイル」＞「エディター」 2. 「スペースを使用したインデント」を選択。 * XML･エディタ設定 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「XML」＞「XMLファイル」＞「エディター」 2. 「スペースを使用したインデント」を選択。 * タスク・タグ設定 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「一般」＞「エディター」＞「構造化テキスト・エディター」＞「タスク・タグ」 2. 「タスク・タグの検索を可能にする」をチェック。 * 補完候補を常時表示 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「Java」＞「エディター」＞「コンテンツ・アシスト」 2. 「Javaの自動有効化トリガー」を「.」→「.abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ_」に変更 * ガベージコレクションをON 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「一般」 2. 「ヒープ・ステータスを表示」をチェック。 * JPAの警告非表示 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「Java 永続化」＞「JPA」＞「エラー警告」 2. 「プロジェクト」＞「指定された接続がないプロジェクト」を『情報』に変更 3. 「型」＞「persistence.xml ファイルにリストされていない管理されたクラス」を『情報』に変更 * HTMLの警告非表示 1. 【eclipseメニュー】＞「ウィンドウ」＞「設定」＞「Web」＞「HTMLファイル」＞「エディター」＞「検証」 2. 「要素」＞「無効なタグロケーション」を『無視』に変更 3. 「属性」＞「未定義の属性名」「未定義の属性値」を『無視』に変更 ## ランチャーのカスタマイズ ------------------------------------------------------------------------------- デスクトップ下のランチャーに下記のコマンドを追加する。 * eclipse起動 -

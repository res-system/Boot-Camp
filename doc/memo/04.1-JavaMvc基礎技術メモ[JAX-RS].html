# JavaMvc基礎技術メモ[JAX-RS] ※コードの表示がおかしい場合は、ソースの表示でコードを確認してください。 ## JAX-RS 2.0 (Jersey2) Webアプリケーション構築 ------------------------------------------------------------------------------- 1. プロジェクト「HelloWorldJersey」を作成する。 1. 【eclipseメニュー】＞「ファイル」＞「新規」＞「動的 Web プロジェクト」で作成する。 2. 【動的 Web プロジェクト】の「プロジェクト名」に『HelloWorldJersey』を入力。 3. 【ビルド・パス上のソース・フォルダー】を『src』(削除)→『src/main/java』に変更し、次へ。 4. 「web.xmlデプロイメント記述子の生成」をチェックし、 「コンテンツ・ディレクトリー」を『WebContent』→『src/main/webapp』に変更して、完了。 2. Gradleプロジェクトへ変換 1. build.gradleファイルを作成する. ```build.gradle apply plugin: 'java' apply plugin: 'war' sourceCompatibility = 1.8 targetCompatibility = 1.8 repositories { jcenter() } /* war化する際に梱包されたくない(参照のみ)ライブラリは、providedCompileで指定. */ dependencies { // servlet. providedCompile('javax.servlet:javax.servlet-api:3.+') // JAX-RS実装(jersey). compile("org.glassfish.jersey.containers:jersey-container-servlet:2.+") compile("org.glassfish.jersey.inject:jersey-hk2:2.+") // JAX-RS実装(jersey(json拡張)). compile("org.glassfish.jersey.media:jersey-media-json-jackson:2.+") // ./main/lib配下の全てのjarファイルをコンパイル時のクラスパスに追加 compile fileTree(dir: './src/main/webapp/WEB-INF/lib', includes: ['*.jar']) // junit testCompile 'junit:junit:4.+' } ``` 2. 【プロジェクト右クリックメニュー】＞「構成」＞「Convert to Gradle(STS) Project」を選択します。 3. Gradleのリフレッシュ * 【プロジェクト右クリックメニュー】＞「Gradle(STS)」＞「依存関係のリフレッシュ」を選択します。 * 【プロジェクト右クリックメニュー】＞「Gradle(STS)」＞「すべてリフレッシュ」を選択します。 3. web.xmlの作成。 ```/src/main/webapp/WEB-INF/web.xml Hello World Jersey index.html index.htm index.jsp default.html default.htm default.jsp Hello World Jersey org.glassfish.jersey.servlet.ServletContainer jersey.config.server.provider.packages sample Hello World Jersey /restful/* ``` 4. Servlet(sample.Example)の作成。 ```/src/main/java/sample/Example.java package sample; import javax.ws.rs.GET; import javax.ws.rs.Path; import javax.ws.rs.PathParam; import javax.ws.rs.Produces; import javax.ws.rs.core.MediaType; @Path("hello/{name}") public class Example { @GET @Produces({MediaType.TEXT_PLAIN}) public String sayHello(@PathParam("name") String name) { return "hello," + name + "!"; } } ``` 5. 実行する。 1. 【Gradleタスクウィンドウ】を開く。 2. プロジェクトを「HelloWorldJersey」に設定。 3. 「build」をクリック。 4. /build/libs/HelloWorldJersey.warが作成されているので、 /opt/lampp/tomcat8/webapps/へwarをコピーする。 5. Tomcat起動。 6. サーバの起動後、ブラウザで確認する - http://localhost:8080/HelloWorldJersey/restful/hello/Jersey
